Follow Steps to use scripts:

1. Upload all scripts to your Linux Image
2. Run sudo chmod +x chmodder.sh (this will make this script executable)
3. Run sudo ./chmodder.sh (this will make ALL scripts executable)
4. Run sudo ./MasterScript.sh (you have activated the master script)
5. Pick a number that corresponds with the function you want to execute
6. For Critical Srvices please use option 6
7. Press the number corresponding to the services you want to configure
8. TO go back to master script press 0 
9. If you want to re-run then stay in the main loop by entering another number
10. To exit just select the exit number
